# Sistema de Gerenciamento de notas e frequência

`Maria Eduarda Botelho`

## Descrição

Sistema criado para cadastrar alunos, diciplinas, frequência e notas. Em seu mecanismo mede-se as médias de cada matéria e a cada aluno acima dela é listado embaixo. Outralistagem é alunos com 
frequência abaixo de 75% são listados.
